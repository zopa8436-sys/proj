Task1 automated solution (no manual input)

Build and run (Linux):

```bash
cd Сойкин_Святослав/Task1
g++ -std=c++17 -O2 main.cpp VectorTask1.cpp -o task1
./task1
```

The program generates random `n` in [12,30] and a random interval `a<=b`, saves the generated vector to `vector_before.bin`, performs the requested operations and prints results.
